# 손해노노
### 소개
[SSAFY 8기 1학기] 프로젝트로 제작하게 됐습니다. 부동산 거래정보, 주변시세, 상권을 알아봐주는 손해노노입니다.

### 참여자
이동준, 박태환

### 역할
기능단위로 분리해 제작했어서 따로 FE/BE 나누지 않고 함께 작업했습니다.

### 환경
[BE] SpringBoot 2.7, Mybatis, AWS RDS

[FE] Vue 2
